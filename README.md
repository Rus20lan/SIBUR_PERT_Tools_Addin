# SIBUR PERT Tools Addin

Профессиональная надстройка (Add-in) для Microsoft Project, предназначенная для автоматизации оценки длительности задач по методу **PERT** (Program Evaluation and Review Technique). Разработано специально для повышения точности планирования графиков в среде SIBUR.

## 📌 Основные функции

* **Автоматический расчет PERT**: Расчет средневзвешенной длительности на основе трех оценок: Оптимистической (O), Наиболее вероятной (M) и Пессимистической (P).
* **Гибкая область применения**: Возможность применить расчет как ко всем задачам проекта, так и только к выделенным строкам.
* **Интеграция с MS Project**: Использование стандартных полей (Duration1, Duration2, Duration3) для ввода данных и Duration7 для вывода результата.
* **Умное обновление**: Снятие флага "Оценочная длительность" (Estimated) при записи результата, что исключает появление знаков вопроса (?) в столбце длительности.
* **Статус исполнения**: Автоматическая отметка "Применено" в поле Text30 для контроля обработанных задач.

## 🧮 Методология расчета

Надстройка использует классическую формулу PERT (бета-распределение):
**Результат = (Оптимистичная + 4 * Вероятная + Пессимистичная) / 6**

* **Оптимистичная (O)**: Поле `Duration1`
* **Вероятная (M)**: Поле `Duration2`
* **Пессимистичная (P)**: Поле `Duration3`
* **Итоговый расчет**: Записывается в `Duration7` и применяется к основной `Duration`.

## 🚀 Установка (Партизанский метод)

Если в вашей компании ограничена установка `.exe` файлов, используйте метод регистрации через реестр Windows.

1.  **Подготовка файлов**: Скопируйте содержимое папки `Release` в локальную директорию на вашем компьютере (рекомендуется `C:\SIBUR_Tools\PERT\`).
2.  **Регистрация**: Создайте файл `install.reg` со следующим содержимым (укажите ваш путь):

```reg
Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Office\MS Project\Addins\SIBUR_PERT_Tools_Addin]
"Description"="Инструмент расчета оценок по методу PERT для СИБУР"
"FriendlyName"="SIBUR PERT Tools"
"LoadBehavior"=dword:00000003
"Manifest"="C:\\SIBUR_Tools\\PERT\\SIBUR_PERT_Tools_Addin.vsto|vstolocal"
